# Car Management System (Hibernate CRUD)

Java project demonstrating Hibernate CRUD operations for a Car entity.

- **Project Name**: carmanagementsystem  
- **Package**: carcrudhibernate  
- **Entity Class**: Car  
- **Main Class**: CarCrud  

## Features
- Create → persist()
- Read → find(), createQuery()
- Update → merge()
- Delete → remove()

## How to Run
1. Configure `persistence.xml` with your DB connection.
2. Create the `Car` table in your database.
3. Run `CarCrud.java` → menu-driven CRUD operations.
